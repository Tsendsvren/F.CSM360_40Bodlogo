### Down to Zero II /Төрөл: Data Structures, Түвшин: Hard/

Танд n гэсэн бүхэл тоо өгөгдөнө. Та энэ тоог 0 болгох шаардлагатай бөгөөд зөвхөн дараах 2 үйлдлийн аль нэгийг нэг удаа хийж чадна:

n = n - 1 → тоог 1-ээр бууруулах

Хэрвээ n = a × b (энэ үед a ≥ b > 1) бол n-ийг max(a, b) болгож сольж болно

Зорилго: n-ийг 0 болгоход шаардлагатай хамгийн бага алхмын тоог олох

Оролт:

Эхний мөр: q — асуултын тоо

Дараагийн q мөр бүрт: n (1 ≤ n ≤ 10⁶)

Гаралт:

Тус бүр мөрөнд: n-ийг 0 болгоход шаардагдах хамгийн бага алхам
